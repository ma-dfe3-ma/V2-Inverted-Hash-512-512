# V2-Inverted-Hash-512-512
We present de V2 Inverted Hash 512×512.This design scales the Inverted pyramid architecture to 512-byte operation, doubling the Sacred Pair K2 active region from 127 to 255 anchors per permutation round, while preserving the tip-to-base anchor order. The central finding is the discovery of superlinear SAC scaling, the system achieves 1.12%.
